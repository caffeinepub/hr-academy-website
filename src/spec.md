# Specification

## Summary
**Goal:** Remove the duplicate “Contact” navigation link from the header while keeping a single, clear Contact call-to-action.

**Planned changes:**
- Update the desktop header navigation to remove the separate text link labeled “Contact” when the “Contact Us” button is present.
- Update the mobile slide-out (sheet) menu to remove the separate “Contact” link that duplicates the “Contact Us” button.
- Ensure /contact remains reachable via the remaining “Contact Us” header call-to-action.

**User-visible outcome:** Visitors see only one Contact option in the header (the “Contact Us” button) on both desktop and mobile, without any duplicate “Contact” link in navigation menus.
